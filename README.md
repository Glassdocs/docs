# Glassdocs product documentation

The complete documentation for [Glassdocs](https://glassdocs.site) — published at
**https://docs.glassdocs.site**.

This repo is itself a Glassdocs knowledge base: Markdown in `docs/` plus
`mkdocs.yml`, built with Zensical and deployed by the
[Glassdocs publisher](https://github.com/Glassdocs/publisher) with the explicit
`public: true` opt-in (world-readable, no access gate). It doubles as a live,
inspectable example of what a Glassdocs KB looks like — including the
`source-repo` meta tag that lets the Docs Chat extension answer questions about
these pages and propose edits back here.

- **Read the docs:** https://docs.glassdocs.site
- **Start your own KB:** use the template at
  https://github.com/Glassdocs/kb-template/generate — not this repo
- **Admin dashboard:** https://app.glassdocs.site/admin/

Corrections welcome — open an issue or PR, or use the Docs Chat extension's
Edit mode on the published site.
